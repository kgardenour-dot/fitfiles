If containerURL = nil in BOTH targets after trying everything:

1. Go to: https://developer.apple.com/account/resources/identifiers/list
2. Find your App ID (com.banditinnovations.fitlinks)
3. Click Edit
4. Verify "App Groups" capability is ENABLED
5. Click "Configure" next to App Groups
6. Verify "group.com.banditinnovations.fitlinks" is CHECKED
7. Save changes
8. Go to Provisioning Profiles
9. DELETE old profiles for both app and extension
10. Go back to Xcode
11. Xcode will auto-generate new profiles
12. Clean build and test again
